#### 1. Список фильмов
`/movies`
#### 2. Список фильмов с фильтром по жанру
`/movies/by?genre={genre}`
#### 3. Список фильмов с фильтром по году выпуска
`/movies/by?year={year}`
#### 4. Список фильмов с фильтром по режиссёру выпуска
`/movies/by?director={director}`
#### 5. Информация о фильме
`/movies/{id}`
